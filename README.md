# healthcare-cost-prediction
Previsão de Custos Assistenciais e Otimização de Sinistralidade na Saúde Suplementar:  Desenvolvimento de um modelo preditivo de IA e Machine Learning focado na previsão de custos hospitalares (Billing Amount) para otimização do índice de sinistralidade e gestão de risco epidemiológico na saúde suplementar.
